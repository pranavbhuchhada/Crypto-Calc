# Crypto-Calc

A calculator for cryptographic calculation.

##  Features
* Encryption
* Decryption
* Hashing
* Encoding/Decoding
* Mathematical functions

## Tools:

### Symmetric Algorithms:
* Caesar Cipher
* Multiplicative Cipher
* Affine Cipher
* Auto-Key Cipher
* Play-fair Cipher
* Vigenere Cipher
* Hill Cipher

### Asymmetric Algorithms:
* Diffie Hellman key exchange
* RSA

### Hashing Algorithms:
* md5
* sha1
* sha224
* sha256
* sha384
* sha512


### Mathematics tool:
* GCD (Greatest Common Divisor )
* Modulus operator
* Prime Test
* Modular Inverse
* Primitive Root
* Chinese Remainder Theorem
